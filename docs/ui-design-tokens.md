# 肤联智诊 UI Design Token

## 品牌气质
- 用户端：Apple Health 式轻医疗科技感，深色底、绿色医疗强调色、大圆角、毛玻璃。
- Web 端：沉稳专业、信息密度高，强调可读性、筛选效率与状态反馈。

## 用户端 Token
- 页面背景：`#08111F`
- 卡片底色：`rgba(16, 24, 42, 0.78)`
- 主品牌色：`#37E28F`
- 辅助品牌色：`#5F7CFF`
- 风险色：`#FF6B7D`
- 主文字：`#EDF3FF`
- 次文字：`rgba(237, 243, 255, 0.70)`
- 圆角：`32rpx / 44rpx`
- 阴影：`0 28rpx 60rpx rgba(0, 0, 0, 0.30)`

## Web 管理端 Token
- 背景：`#0C1220`
- 面板底色：`rgba(17, 24, 39, 0.84)`
- 软边框：`rgba(255, 255, 255, 0.08)`
- 主文字：`#EFF4FF`
- 次文字：`rgba(239, 244, 255, 0.72)`
- 品牌绿：`#37E28F`
- 品牌蓝：`#5F7CFF`
- 风险红：`#FF6B7D`
- 面板圆角：`28px`
- 阴影：`0 24px 60px rgba(0, 0, 0, 0.28)`

## 组件约束
- 核心按钮统一使用绿色渐变，承载“立即问诊、保存、提交”等主要操作。
- 风险状态统一映射为 `LOW / MEDIUM / HIGH` 的绿、橙、红三档展示。
- 用户端卡片统一使用玻璃质感和柔和描边，避免默认白底组件破坏整体调性。
- Web 端统计优先使用轻量 CSS 图形，降低本地启动复杂度。
